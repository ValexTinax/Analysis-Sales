# Analysis-Sales
On this Sales Project I m used Excel functionality for show the data of coffee sales in the time.

Start with Dataset of Product and Customers;

ORDERS sheet
contains Customers Orders;
Function used:

VLOOKUP; IF; INDEX; MATCH to find the value corresponding to other sheets;

PIVOT_SLICER sheet:
build a Pivot Table and pivot chart for display the Sales of every Coffee type in years;
Timeline for choose the exact time to display;
Slicer to have more interesting filter to select;

COUNTRY BAR CHART:
Pivot table and chart to display the Sales by Country;

TOP 5 CUSTOMERS:
Pivot table and chart to display the Top 5 Customers;

DASHBOARD:
Is the presentation that contains all data extract from other files.
